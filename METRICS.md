# Metrics

## Activation
- Audit start rate = audits_started / landing_page_sessions
- Audit completion rate = audits_completed / audits_started

## Conversion
- Email capture rate = emails_captured / audits_completed
- Save/share rate = audits_saved / audits_completed

## Engagement
- Return visits = returning_sessions / total_sessions
- Result link views per audit = total_result_views / audits_saved

## Revenue
- Paid conversion rate = paid_accounts / active_accounts
- ARPA = monthly_revenue / paid_accounts

## Quality
- Recommendation acceptance rate (manual survey)
- False positive rate (manual review)
